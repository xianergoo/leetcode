## 项目结构

```
.
├── README.md           # 题目大纲与进度追踪

├── notes/              # 分类学习笔记
│   ├── 01-array.md
│   ├── 02-string.md
│   └── ...
├── solutions/          # 题解代码（按分类组织）
│   ├── array/          # 数组
│   ├── string/         # 字符串
│   ├── linked_list/    # 链表
│   └── ...

    └── types.go
```
