# 学习流程

## 每道题的 5 步学习法

| Step | 阶段 | 说明 |
|------|------|------|
| 1 | 专题笔记 | 该分类的核心知识点、常见套路、Go 语言要点 |
| 2 | 审题思考 | 给出题目描述，你先独立思考 3-5 分钟 |
| 3 | 思路讲解 | 逐步讲解解法思路、关键点、边界情况 |
| 4 | 代码实现 | 完整 Go 代码 + 测试用例 + 复杂度分析 |
| 5 | 变式追问 | 1-2 个验证性问题，确认你真正理解了 |

## 专题顺序

按推荐学习路线：哈希表 → 双指针 → 滑动窗口 → 链表 → 二叉树 → 回溯 → 图论 → DP → 其他

## 做题原则

1. **先思考再看答案** — 哪怕想不出来，思考过程本身就在建立神经连接
2. **手写代码** — 看懂 ≠ 会写，务必自己敲一遍
3. **关注复杂度** — 每道题都要能说清时间/空间复杂度
4. **不赶进度** — 一天 1-3 题即可，重在消化
